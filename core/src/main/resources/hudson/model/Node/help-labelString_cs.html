Štítky (nebo značky) se používají pro přiřazení mnoha agentů do jedné logické skupiny.

Například, máte-li mnoho strojů s Windows a úlohu, která na tomto systému musí běžet, pak můžete jim všem přidat štítek windows a pak svázat úlohu s tímto štítkem.
Tímto zajistíte, že vaše úloha poběží na agentech s Windows a vyloučíte ty, co jej nemají.

Štítky nemusí nezbytně označovat operační systém agenta; můžete je také použít pro označení architektury procesoru, případně že má k dispozici určitý nástroj.

Při zadávání více štítků najednou je třeba oddělit mezerou. Například, windows docker přidělí agentovi dva štítky: windows a docker.

Śtítky mohou obsahovat všechny znaky, kromě mezer, ale některých byste se měli vyvarovat, jako například těchto: !&|<>(), protože jiné funkce Jenkins umožňují štítky určovat pomocí výrazů, kde mohou být tyto znaky použity.
