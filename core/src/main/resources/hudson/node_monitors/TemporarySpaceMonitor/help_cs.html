Tento monitor sleduje dostupné místo v dočasném adresáři a pokud klesne pod určitou hodnotu, bude tento uzel odstaven.

V tomto adresáři nástroje Java a testy/sestavení často vytvářejí soubory, a bez dostatku volného místa nelze zajistit jejich správné fungování.

Abychom to přiblížili, jedná se o kontrolu dostupného místa diskového oddílu, kde je umístěna složka určená systémovou vlastnosti java.io.tmpdir. Abyste zjistili, kde se složka v rámci agenta nachází, navštivte ${rootURL}/computer/SLAVENAME/systemInfo.
