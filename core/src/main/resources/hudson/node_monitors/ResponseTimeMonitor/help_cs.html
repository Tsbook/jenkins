Tento monitor měří dobu odezvy zpráv odeslaných od hlavního serveru agentovi a zpět a pokud bude opakovaně delší, než zadaný limit, agenta odstaví.

To je užitečné pro zjišťování neodpovídajících agentů, či jiných problémů v síti, které ucpávají komunikační kanály. Konkrétněji, hlavní server odešle agentovi příkaz bez funkce, a spočítá za jak dlouho od agenta dostane výsledek této operace.
