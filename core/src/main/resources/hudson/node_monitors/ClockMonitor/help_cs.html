Tento monitor sleduje časový rozdíl mezi hlavním serverem a uzly. Přestože samotný Jenkins je obecně schopen snášet časové rozdíly mezi systémy, činnosti systémů pro správy verzí a sdílené přístupy k souborům (třeba NFS, sdílení souborů Windows) mohou vykazovat zvláštní problémy, pokud zapojené systémy mají odlišný čas.

Abyste měli čas vždy synchronizovaný, přečtěte si více o protokolu NTP.
