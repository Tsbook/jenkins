Pokud je váš server Jenkins umístěn za firewallem a nemá přímý přístup k internetové síti, a pokud je váš server JVM špatně nastaven (Pro další informace si přečtěte vlastnosti JDK měnící síťová připojení) a nemá povoleno připojení k internetu, můžete v tomto poli zadat název HTTP proxy serveru, aby Jenkins mohl instalovat zásuvné moduly vaším prostřednictvím. Nezapomeňte, že při komunikaci s aktualizačním centrem pro stažení modulů využívá Jenkins protokol HTTPS.

Pokud toto pole ponecháte prázdné bude se Jenkins pokoušet připojit k internetu přímo.

Pokud nevíte, jakou hodnotu zde zadat, zkontrolujte nastavení proxy vašeho prohlížeče.
