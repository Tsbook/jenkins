Použít servletový kontejner pro ověřování uživatelů, dle jeho specifikace. Takto se historicky postupovalo až do Jenkins verze 1.163. Hodí se hlavně v následujících situacích:

  2. Váš kontejner již je nastaven příslušnou oblastí zabezpečení a jednoduše chcete, aby ji Jenkins používal. (V některých případech kontejnery mají lepší dokumentaci nebo vlastní způsob jak se připojit ke konkrétním oblastem uživatelů.)
