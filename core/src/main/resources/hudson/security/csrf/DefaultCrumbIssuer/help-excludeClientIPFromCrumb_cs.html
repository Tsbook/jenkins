Některé HTTP proxy filtrují údaje, které výchozí vydavatel ověřovacích známek využívá k výpočtu jednorázové kontrolní hodnoty. Pokud je na cestě mezi vaším prohlížečem a Jenkins použita HTTP proxy a při odesílání formuláře v Jenkins dostanete chybu 403, může zaškrtnutí této volby pomoci. Tímto bude kontrolní hodnota snadněji padělatelná.
