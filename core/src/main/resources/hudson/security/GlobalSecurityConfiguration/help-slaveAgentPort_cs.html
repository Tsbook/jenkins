Jenkins používá port TCP pro komunikaci s agenty spuštěnými pomocí JNLP. Standardně je port vybírán náhodně, aby se předešlo konfliktům, ale to velmi ztěžuje jeho případné zabezpečení. Pokud agenty spouštěné přes JNLP nepoužíváte, je nejlepší tento port zakázat. Případně lze nastavit konkrétní číslo portu, abyste následně mohli nastavit svůj firewall.
