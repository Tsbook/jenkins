Při zabezpečení Jenkins máte možnost nastavit způsoby ověření (jak se mohou lidé identifikovat) a oprávnění (které oprávnění dostanou).

Mnoho možností je součástí základní verze. Je důležité si uvědomit, že udělení důležitých oprávnění anonymním uživatelům, nebo umožnění uživatelům přihlásit se a oprávnění udělovat všem přihlášeným uživatelům, nebude automaticky znamenat větší bezpečnost.

Další informace o zabezpečení a Jenkins si přečtěte tento dokument.
